Instructions for use:

  • This program is called SkyboundDB 2.0.
  • Its main purpose is to calculate the frame advantage of situations in the game Granblue Fantasy: Versus.
  • The dealer is the player that has dealt a certain move, and the responder is the player who is assumed to have blocked said move.
  • On the left choose the move that the dealer will perform.
  • On the right choose the move the responder will perform as a response to blocking the stated dealer move.
  • The program can also check for ALL moves in the dealers character kit in response to the responder's move.
  • Press Submit to get the results from the selected moves.
  • The program will return the result if said move can be punished or not from the responder's perspective.
  • It will return how minus a move is, and how much time the responder has to perform the stipulated action to punish.
  • If ALL is selected, it will return a list of all possible moves that can be punished using the responder's move.
  • You can also reset back to this screen by pressing Reset in a result page.
  • Enjoy the program!
